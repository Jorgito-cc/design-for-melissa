# 💜 Regalo para Meli

Una tarjeta interactiva diseñada especialmente para el cumpleaños de Melissa, con temática de **BTS** y un estilo limpio para una diseñadora.

## ✨ ¿Qué incluye?
- **Botón Sorpresa:** Un regalo animado que revela la carta al hacer clic.
- **Diseño Responsive:** Se ve perfecto tanto en celular como en computadora.
- **Estética Army:** Detalles en color lila, el logo de BTS y el lema *Apobangpo*.

## 🚀 Tecnologías
- React
- Tailwind CSS

## 🛠️ Cómo verlo
1. Instala las librerías con `npm install`.
2. Corre el proyecto con `npm run dev`.
3. Para subirlo a Netlify, usa `npm run build` y sube la carpeta `dist`.
